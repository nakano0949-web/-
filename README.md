# 輪作スケジュール生成アプリ (v2.0)

有限幾何のブロックデザインを用いた、科学的で持続可能な輪作計画を立てるためのツールです。

## 特徴
- **数学的アプローチ**: アフィン平面 $AG(2,p)$ や射影平面 $PG(2,p)$ を利用して、全区画の組み合わせを網羅。
- **地力収支解析**: 各野菜の窒素負荷を計算し、土壌が肥えるか痩せるかを判定。
- **専門知識の集約**: 20種類以上の科ごとの特徴や相性データを搭載。

## 使い方
1. `index.html` をブラウザで開く（GitHub Pagesで公開中）。
2. 平面タイプと素数 $p$ を選択。
3. 各区画に育てたい野菜を割り当てる。
4. 「スケジュール生成」を押して、年ごとの計画を確認！

## 技術スタック
- HTML5 / CSS3
- JavaScript (Vanilla JS)
